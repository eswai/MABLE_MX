# MABLE

MABLEはワイヤレス、左右分割、36キーのコンパクトなキーボードです。
キーボードカレンダー2024年3月に掲載されました。

* MiniAxe互換キーレイアウト
* 省電力なnRF52840採用
* Bluetooth接続、電池内蔵の完全ワイヤレス
* USB-Cからバッテリー充電 (リチウムポリマー電池の取り扱いにはご注意ください)
* ZMK firmwareもしくはKMK firmware
* 3Dプリンタ製のケース
